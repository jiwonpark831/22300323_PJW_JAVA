제목: 영화 리스트 CRUD 프로그램<br>
제작의도: 영화를 좋아하고 자주봐서 영화 CRUD 프로그램을 만들게 되었다. 영화의 제목, 평점, 누적 관객수, 개봉년도를 정리하였고 필수 기능과 추가로 평점순, 누적관객수순으로 정렬하는 sort기능을 구현하였다.<br>
소감: 미니 프로젝트를 평소에 관심 있는 주제와 연관 지어서 할 수 있어서 의욕을 가지고 재미있게 할 수 있었다. CRUD를 여러 번 반복하니 확실히 익숙해진 것이 느껴졌고 미니 프로젝트의 결과물을 실행해 보며 이번 학기 시작할 때의 나를 떠올려보았는데 그때와 비교해 봤을 때 코딩 실력이 많이 발전했다고 생각해서 뿌듯했다. 하지만 아직 인터넷 서치 없이 완벽한 프로그램을 짜는 것은 조금 힘들어서 방학 때 더 많은 문제를 풀어봐야겠다고 다짐했다.<br>
